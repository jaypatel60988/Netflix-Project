Netflix Data Analysis Dashboard 📊

📌 Project Overview
This project focuses on analyzing Netflix dataset using Tableau to uncover insights about content distribution, genres, and trends.

🛠 Tools Used
- Tableau
- Excel / CSV Dataset

📂 Dataset
- Netflix Movies and TV Shows dataset

📈 Key Responsibilities
- Cleaned and prepared dataset for analysis
- Imported data into Tableau for visualization
- Designed interactive dashboards to analyze content trends
- Analyzed distribution of Movies vs TV Shows
- Explored content by genre, country, and release year
- Created filters and dynamic visuals for better user interaction

📊 Key Insights
- Movies dominate the platform compared to TV Shows
- Content production increased significantly after 2015
- Certain genres like Drama and Comedy are most popular
- USA has the highest content contribution
